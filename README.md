# Contract Comparator

This is a simple Streamlit app for comparing multiple contracts against a reference document.

## Features

- Upload a reference contract and up to 20 documents (`txt` or `docx`).
- See a grid of similarity ratios per clause.
- Expand any clause to view the exact wording from each document.

## Usage

Install dependencies and run the Streamlit app:

```bash
pip install -r requirements.txt
streamlit run app/app.py
```

Upload your reference file and the documents you want to compare.
